# LoadmapAI - Product Requirements Document (PRD)

## 1. 제품 개요

### 1.1 제품명
**LoadmapAI** - AI 기반 학습 로드맵 생성 및 관리 플랫폼

### 1.2 비전
목표만 입력하면 AI가 월별/주별/일별 학습 계획을 자동으로 생성하고, 퀴즈를 통해 학습 효과를 극대화하는 플랫폼

### 1.3 타겟 사용자
- 새로운 기술을 체계적으로 학습하고 싶은 개발자
- 자격증/시험 준비를 하는 학습자
- 장기 목표를 체계적으로 관리하고 싶은 사람

## 3. 사용자 스토리

### 3.1 회원가입/로그인
- 사용자는 이메일로 회원가입할 수 있다
- 사용자는 Google/GitHub 계정으로 소셜 로그인할 수 있다
- 사용자는 비밀번호를 재설정할 수 있다

### 3.2 로드맵 생성
- 사용자는 목표를 입력하고 모드(플래닝/러닝)를 선택할 수 있다
- 사용자는 기간(1-6개월)과 하루 학습시간을 설정할 수 있다
- AI가 월별 마일스톤 → 주별 태스크 → 일별 태스크를 자동 생성한다

### 3.3 로드맵 관리 (플래닝 모드)
- 사용자는 월별/주별/일별 뷰를 드릴다운으로 탐색할 수 있다
- 사용자는 일별 태스크를 체크박스로 완료 표시할 수 있다
- 진행률이 자동으로 계산되어 표시된다

### 3.4 학습하기 (러닝 모드)
- 사용자는 일별 학습 주제의 [학습 시작하기] 버튼을 클릭한다
- AI가 해당 주제의 퀴즈(5-10문제)를 자동 생성한다
- 사용자는 객관식/단답형/서술형 문제를 풀 수 있다
- AI가 답변을 채점하고 피드백을 제공한다
- 오답 문제만 다시 풀어볼 수 있다

### 3.5 대시보드
- 사용자는 오늘의 할 일 목록을 볼 수 있다
- 사용자는 전체 진행률과 학습 통계를 확인할 수 있다
- 사용자는 연속 학습일(streak)을 확인할 수 있다

## 4. 기술 사양

### 4.1 기술 스택

| 영역 | 기술 |
|------|------|
| Backend | FastAPI, SQLAlchemy, Alembic, Pydantic |
| Frontend | React 18, TypeScript, Vite, TailwindCSS, Zustand |
| AI | LangGraph, LangChain, Claude API (Anthropic) |
| Database | PostgreSQL 15 |
| Cache | Redis 7 |
| Auth | JWT, OAuth2 (Google, GitHub) |
| Deploy | Docker, Docker Compose |

### 4.2 데이터 모델 요약

```
User
  └── Roadmap (mode: planning|learning)
        └── Milestone (월별)
              └── WeeklyTask (주별)
                    └── DailyTask (일별)
                          └── Quiz (러닝 모드만)
                                └── Question
                                      └── UserAnswer
```

### 4.3 AI 워크플로우 (LangGraph)

1. **로드맵 생성 그래프**: Goal Analyzer → Roadmap Planner → Milestone/Task Generator → Validator
2. **문제 생성 그래프**: Topic Analyzer → Question Generator (객관식/단답형/서술형) → Quality Validator
3. **채점 그래프**: Answer Analyzer → Feedback Generator → Score Calculator

## 5. 제약 사항

### 5.1 비즈니스 제약
- 로드맵 기간: **최소 1개월 ~ 최대 6개월**
- 퀴즈당 문제 수: 5-10문제
- 문제 유형: 객관식, 단답형, 서술형 (코딩 문제 제외)

### 5.2 기술 제약
- AI API 호출 제한 (Rate Limiting)
  - 로드맵 생성: 5회/일
  - 퀴즈 생성: 20회/일
  - AI 채점: 100회/일

### 5.3 보안 요구사항
- 비밀번호: bcrypt 해싱
- API 인증: JWT Bearer Token
- CORS 설정 필수
- 환경변수로 민감정보 분리

## 7. 릴리즈 계획

### Phase 1: MVP (4주)
- 기본 인증 (이메일 + 소셜 로그인)
- 로드맵 CRUD + AI 생성
- 플래닝 모드 완성 (드릴다운 UI, 태스크 체크)

### Phase 2: 러닝 모드 (2주)
- 퀴즈 시스템 구현
- AI 문제 생성 + 채점
- 결과/피드백 페이지

### Phase 3: 고도화 (추후)
- 오답 복습 기능
- 학습 통계 대시보드
- 모바일 최적화
