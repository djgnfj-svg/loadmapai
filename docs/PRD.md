# LoadmapAI - Product Requirements Document (PRD)

## 1. 제품 개요

### 1.1 제품명
**LoadmapAI** - AI 기반 학습 로드맵 생성 및 관리 플랫폼

### 1.2 비전
목표만 입력하면 AI가 월별/주별/일별 학습 계획을 자동으로 생성하고, 체계적인 진행 관리를 통해 학습 효과를 극대화하는 플랫폼

### 1.3 타겟 사용자
- 새로운 기술을 체계적으로 학습하고 싶은 개발자
- 자격증/시험 준비를 하는 학습자
- 장기 목표를 체계적으로 관리하고 싶은 사람

---

## 2. 핵심 기능

### 2.1 플래닝 모드

| 모드 | 설명 | 핵심 가치 |
|------|------|----------|
| **플래닝 모드** | 일정 관리 중심 | 태스크 체크리스트, 진행률 트래킹, AI 채팅 기반 편집 |

### 2.2 기능 목록

| 기능 | 설명 |
|------|------|
| AI 로드맵 자동 생성 | 목표 입력 시 월/주/일별 태스크 자동 생성 |
| 월→주→일 드릴다운 UI | 계층적 탐색으로 세부 태스크 확인 |
| 일별 태스크 체크 | 체크박스로 완료 상태 관리 |
| AI 채팅 기반 편집 | 자연어로 로드맵 수정 요청 |
| 로드맵 확정/해제 | 편집 완료 후 확정하여 안정적 실행 |
| 진행률 추적 | 자동 계산되는 전체 진행률 |

## 3. 사용자 스토리

### 3.1 회원가입/로그인
- 사용자는 이메일로 회원가입할 수 있다
- 사용자는 Google/GitHub 계정으로 소셜 로그인할 수 있다
- 사용자는 비밀번호를 재설정할 수 있다

### 3.2 로드맵 생성
- 사용자는 학습 목표를 입력할 수 있다
- 사용자는 기간(1-6개월)과 하루 학습시간을 설정할 수 있다
- AI가 월별 목표 → 주별 태스크 → 일별 태스크를 자동 생성한다

### 3.3 로드맵 관리
- 사용자는 월별/주별/일별 뷰를 드릴다운으로 탐색할 수 있다
- 사용자는 일별 태스크를 체크박스로 완료 표시할 수 있다
- 진행률이 자동으로 계산되어 표시된다

### 3.4 로드맵 편집
- 사용자는 AI 채팅을 통해 로드맵을 자연어로 수정할 수 있다
- 사용자는 태스크를 직접 추가/수정/삭제할 수 있다
- 사용자는 로드맵을 확정하여 편집을 잠글 수 있다

### 3.5 대시보드
- 사용자는 오늘의 할 일 목록을 볼 수 있다
- 사용자는 전체 진행률과 학습 통계를 확인할 수 있다
- 사용자는 연속 학습일(streak)을 확인할 수 있다

## 4. 기술 사양

### 4.1 기술 스택

| 영역 | 기술 |
|------|------|
| Backend | FastAPI, SQLAlchemy, Alembic, Pydantic |
| Frontend | React 18, TypeScript, Vite, TailwindCSS, Zustand |
| AI | LangGraph, LangChain, Claude API (Anthropic) |
| Database | PostgreSQL 15 |
| Cache | Redis 7 |
| Auth | JWT, OAuth2 (Google, GitHub) |
| Deploy | Docker, Docker Compose |

### 4.2 데이터 모델 요약

```
User
  └── Roadmap
        └── MonthlyGoal (월별 목표)
              └── WeeklyTask (주별 태스크)
                    └── DailyTask (일별 태스크)
        └── RoadmapConversation (AI 채팅 기록)
```

### 4.3 AI 워크플로우 (LangGraph)

1. **로드맵 생성 그래프**: Goal Analyzer → Monthly Generator → Weekly Generator → Daily Generator → Validator → Saver
2. **로드맵 편집 그래프**: Chat Message Parser → Change Proposal Generator → Change Applier

## 5. 제약 사항

### 5.1 비즈니스 제약
- 로드맵 기간: **최소 1개월 ~ 최대 6개월**
- 로드맵 확정 후 편집 가능 (편집 횟수 추적)

### 5.2 기술 제약
- AI API 호출 제한 (Rate Limiting)
  - 로드맵 생성: 5회/일
  - AI 채팅 편집: 50회/일

### 5.3 보안 요구사항
- 비밀번호: bcrypt 해싱
- API 인증: JWT Bearer Token
- CORS 설정 필수
- 환경변수로 민감정보 분리

## 7. 릴리즈 계획

### Phase 1: MVP (완료)
- 기본 인증 (이메일 + 소셜 로그인)
- 로드맵 CRUD + AI 생성
- 플래닝 모드 (드릴다운 UI, 태스크 체크)

### Phase 2: AI 편집 (완료)
- AI 채팅 기반 로드맵 편집
- 로드맵 확정/해제 기능
- 스케줄 설정 기능

### Phase 3: 고도화 (추후)
- 학습 통계 대시보드
- 모바일 최적화
- 공유 기능
